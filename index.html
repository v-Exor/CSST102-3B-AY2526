CSST 102: Basic Machine Learning

Introduction

Basic Machine Learning introduces students to the fundamental concepts, principles, and techniques of machine learning. The course explores how computers can learn from data, recognize patterns, and make predictions or decisions with minimal human intervention. Students will gain hands-on experience with basic algorithms, data preprocessing, and model evaluation, laying the groundwork for more advanced studies in artificial intelligence and data science.

The subject emphasizes both the theoretical understanding and practical applications of machine learning across various fields such as business, science, and technology.

Key Concepts

1. Supervised and Unsupervised Learning

Covers the two main types of learning: supervised learning (training with labeled data) and unsupervised learning (finding patterns in unlabeled data).

2. Model Training and Evaluation

Focuses on building machine learning models, training them on datasets, and evaluating their performance using metrics like accuracy, precision, and recall.

3. Data Preprocessing and Feature Engineering

Highlights the importance of cleaning, transforming, and selecting relevant data to improve the accuracy and efficiency of machine learning models.
